# 朗朗口腔

* 技术栈：JavaScript、PHP、Scss

## 介绍

- 这是学习 HTML、CSS、JS 后产出的一个小demo
- 根据设计稿还原出真实网页
- 利用 Koala 编译 Scss 代码；利用 phpstudy 来访问网页
- 设计稿存放在 images 文件夹
